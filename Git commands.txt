just add commit then push

1) git add -A
2) git commit -m "you message what save you have done in this commit"
// git commit -a -m " message " ( this can be used to direct commit changes without adding it to staging )
3) git push -u origin master 
// git push ( after setting origin master direct use git push to upload the code to the github )
